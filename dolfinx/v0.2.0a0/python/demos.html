Demos¶

Introductory demos¶

These demos illustrate core DOLFINx/FEniCSx usage and are a good way to begin learning FEniCS. We recommend that you go through these examples in the given order.

  1. Getting started: Poisson equation.

  2. Using mixed, Taylor-Hood element: Stokes equation

More advanced demos¶

These examples typically demonstrate how to solve a certain PDE using more advanced techniques. We recommend that you take a look at these demos for tips and tricks on how to use more advanced or lower-level functionality and optimizations.

All documented demos¶

  • Poisson equation
  • Cahn-Hilliard equation
  • Stokes equations with Taylor-Hood elements
  • Elasticity equation
  • Mesh generation using GMSH python API
  • Helmholtz equation
  • Static condensation of linear elasticity
  • Using pyvista for visualization
